Apa itu Decoy?

Decoy = paket dummy yang ditambahin bareng paket utama waktu beli biar transaksi keliatan "wajar". Default dipakai akun PREPAID biasa, Prio dipakai akun PRIORITAS / PRIOHYBRID / GO.

Penting: Tekan tombol Test Fetch di tiap decoy buat verifikasi apakah masih valid sebelum dipakai.
